# Style Example Images

These images are shown in the Style Guide modal as previews for each visual style.
Each depicts: **A person sitting on an SF Muni bus, looking at their phone.**

## How to generate

Go to [Google AI Studio](https://aistudio.google.com) → select **Gemini 2.5 Flash Image** (Nano Banana) → paste each prompt below → download the result → save to this folder at the filename listed.

Resize/crop each to roughly **400×250px** (landscape) for fast loading as thumbnails.
